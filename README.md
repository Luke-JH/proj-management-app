# Project Management App

(Work in Progress) Welcome to my free, open source Project Management tool, which is intended for managing small to medium workflows.

## Tech Stack

### Language

TypeScript

### Frontend

Vue, Nuxt, Tailwind

### Backend

Node, Nest

## How to Run

Naviage into the app's directory on your system.

### Frontend

```bash
cd /frontend &&
npm run dev
```

### Backend

```bash
cd /backend &&
npm run start
```
